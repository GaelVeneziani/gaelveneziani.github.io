University Project / 2021

The idea behind this project was to create a still illustration that would be a "portal" to another world. The still image, with the use of the phone application "Artivive", would then transform into a short animation of 5 seconds, in which the user would enter the world behind the portal.
The idea behind this piece was to create a mental portal, in which we could lose ourselves inside just by looking inside.

Media:
- Adobe Photoshop
- Adobe After Effects
